 2. 기술적 설명
이번 광고 영상은 텍스트 프롬프트를 기반으로 이미지, 영상, 오디오를 각각 생성형 AI로 제작한 뒤 편집 도구에서 하나의 광고 영상으로 통합하였다.
 2.1 제작 과정
 ① 기획 및 스토리보드 작성
· 	브랜드 아이덴티티 설정
· 	타겟 및 핵심 메시지 정의
· 	10초 이내 광고를 위해 4개 씬으로 구성
· 	씬별 목표 메시지·화면 구성·화면 카피 설정
 ② 이미지 생성
씬별 핵심 장면은 ChatGPT의 이미지 생성 기능을 활용하여 제작하였다. 학습 네이토에도 동일한 프롬프트를 입력해 결과를 비교했으나 캐릭터 형태와 3D 스타일이 기획 의도와 달라 최종 이미지로 사용하지 않았다.
· 	여자아이 캐릭터
· 	AI 음악 친구
· 	따뜻한 숲속 음악 교실
· 	귀여운 3D 애니메이션 스타일
캐릭터 외형, 색상, 배경 분위기와 애니메이션 스타일을 유사하게 유지했으며, 기획 의도에 가장 가까운 ChatGPT 이미지를 비디오 제작의 기준 이미지로 사용하였다.
 ③ 비디오 생성
씬별 3D 이미지를 Gemini의 기준 이미지로 업로드해 이미지 기반 애니메이션을 제작하였다. 화면 비율은 16:9, 스타일은 따뜻한 색감의 비사실적 3D 애니메이션으로 설정했다. 캐릭터와 악기 형태를 유지하고 텍스트·로고·불필요한 사물을 생성하지 않도록 제한 조건을 추가하였다.
· 	악기 연주 움직임
· 	떠다니는 음표와 반짝이는 빛
· 	부드러운 카메라 이동
· 	자연스러운 캐릭터 움직임
잘못된 글자가 생성되는 것을 방지하기 위해 영상 생성 단계에서는 텍스트와 로고를 넣지 않았다. 화면 카피와 브랜드명은 CapCut 편집 단계에서 삽입하였다.
 ④ 오디오 생성
Suno에서 실로폰·우쿨렐레·탬버린·부드러운 손뼉 소리의 분위기를 담은 BGM 후보 3곡을 생성하였다. 그중 멜로숲 뮤직의 밝고 따뜻한 분위기와 9.6초 광고 흐름에 가장 잘 어울리는 1곡을 선정하였다.
 ⑤ 통합 편집
· 	씬별 컷 편집
· 	화면 카피와 브랜드명 삽입
· 	영상과 음악의 타이밍 조정
· 	오디오 레벨 조정
· 	MP4(H.264/AAC) 출력
CapCut은 핵심 비주얼이나 오디오를 생성하는 용도가 아니라 생성형 AI 결과물을 통합하는 편집 도구로만 사용하였다.
 2.2 화면 비율 변경
초기에는 모바일 숏폼을 고려해 9:16 세로형 이미지를 제작했으나 과제 규격이 16:9 가로형임을 확인했다. 단순 확대나 크롭 대신 캐릭터와 악기 구도를 유지하면서 좌우 배경을 확장한 16:9 이미지를 다시 생성하고, 이를 기준으로 애니메이션을 제작하였다.
기본 보정 순서: 비율 확인 → 주요 피사체 위치 확인 → 16:9 재생성 → 해상도 확인 → 색감·톤앤매너 확인 → 최종 영상 변환
색감 기준: 파스텔 옐로·라이트 블루·민트·우드 브라운 중심, 따뜻한 자연광, 밝고 안전한 유아 교육 분위기
제작 우선순위: 비용과 속도보다 캐릭터·악기·3D 스타일의 일관성과 최종 품질을 우선
 2.3 생성형 AI 소스 및 저작권 선언
본 광고의 시각 및 청각 소스는 모두 생성형 AI를 이용하여 제작하였다. 직접 촬영 영상과 유료 스톡 소스는 사용하지 않았다. 사용 도구의 생성 콘텐츠 이용 범위를 확인했으며, 실존 인물의 얼굴을 모방한 딥페이크와 혐오·선정성·폭력 등 유해 콘텐츠는 사용하지 않았다.

 3.4 이미지·비디오 도구 비교
동일한 이미지 프롬프트를 ChatGPT와 학습 네이토에 입력하였다. 학습 네이토는 캐릭터 형태와 3D 스타일이 기획 의도와 다르게 표현되었고, ChatGPT는 여자아이·AI 음악 친구·악기·따뜻한 음악 교실을 비교적 안정적으로 표현해 최종 이미지 도구로 선택하였다.
동일한 기준 이미지와 유사한 악기 연주 프롬프트를 Sora 2와 Gemini에 입력하였다. Sora 2는 움직임이 풍부했지만 외형과 악기 동작이 달라지는 경우가 있었고, Gemini는 기준 이미지의 캐릭터·악기·배경을 비교적 안정적으로 유지해 최종 비디오 도구로 선택하였다.
